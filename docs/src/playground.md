# Interactive Playground

The PipTable Playground lets you experiment with the DSL directly in your browser.

<iframe src="../playground/dist/index.html" style="width: 100%; height: 80vh; border: 1px solid #333; border-radius: 4px;"></iframe>

## Features

- **Live Editing**: Write and modify PipTable code with syntax highlighting
- **Example Gallery**: Learn from pre-built examples
- **Instant Feedback**: Run scripts in-browser via WASM
- **Mobile Friendly**: Works on all devices

## Keyboard Shortcuts

- **Run Code**: `Ctrl+Enter` (Windows/Linux) or `Cmd+Enter` (Mac)
- **Format Code**: Coming soon

## Note

The playground runs scripts in-browser. File and network operations (import/export/fetch) are disabled in the browser environment.
